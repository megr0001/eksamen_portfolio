Grundlæggende Web

I temaet Grundlæggende Web lærte vi en masse basis viden om kodning. Her lavede vi vores første kodede site som var vores studiestartsprøve. Vi lærte om alt fra fonte, baggrund til grids og hvordan man brugte det når man kodede en hjemmeside helt fra bunden.

Vi lærte om CSS, der er 3 måder at skrive CSS på

Vi bruger den første, som er et eksternt stylesheet, hvor man linker til den
i sit html site.

  • Eksternt css - Her linkes der til en ekstern css fil
  • Internt css - Her skrives css'en inde i et style element i headeren
  • Inline css - Her skrives css'en som en style attribut inde i et tag

Designprincipper

I det første tema hvor vi lavede studiestartsprøven, lærte vi også om forskellige designprincipper.
På de nedenstående billeder kan man se nogle af disse principper i brug:

  • Hvordan mennesker opfatter objekter og relaterer objekter til hinanden
  • Loven om nærhed/proximity - hvis man fx har objekter der er tætte på hinanden opfatter vi det som at de relaterer til hinanden. Når én tekst er tæt på en anden føler vi at de relaterer til hinanden og hører sammen

Gestalt loven

  • Loven om kontinuitet - vi færdiggør former så vores hjerne danner relationer til ting. Elementer der er placeret så de danner linjer eller kurver opfattes som relateret til hinanden.
  • Loven om lukkethed - når elementer er lukket inde i den samme boks fx så mener vi at de relaterer til hinanden
  • Loven om figurdannelse - elementer der er seperate kan opfattes som en samlet figur, selvom denne figur egentlig ikke er eksplicit- ses mest i logoer

Visuelt hierarki

  • Kontrast
  • Størrelse, postition, farve
  • Spil på hvad der er figurer og hvad der er baggrund
  • Web typografi, med serif / uden serif (fødder)
  • Ting du skal undgå - læselig skrift/paragraffer: der står med fed, kursiv, STORE BOGSTAVER HELE TIDEN etc.
  • Ting du bør gøre - hav forskellige skriftstørrelse men i hierarkisk orden
  • Del paragrafferne op og hav separate overskrifter - loven om nærhed

Farver:

  • Addetive farver og subtraktive farver - RGB (RedGreenBlue) og CMYK (CyanMagentaYellowKey(sort)) - RGB er skærm farver og her er der mange lyse farver. Når man blander alle RGB så får man hvid, men hvis man gør det i virkeligheden får man brun
  • HSL (HueSaturationLightness) - Hue - kulør, Saturation - hvor gennemtrængende den er, Lightness - hvor meget lys der er
  • Komplementær kontrast - farver der ligger overfor hinanden i farvecirklen - harmoniske kontrastfarver. Orange knap på blå baggrund eksempelvis - find komplimentære farver via Adobe Color
  • Kulde/varme kontrast
